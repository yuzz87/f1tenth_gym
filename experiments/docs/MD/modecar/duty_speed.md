# ESC duty比・速度実測データ

## 目的

ESCへ出力するPWM duty比[%]と、実車速度[m/s]の対応関係を記録する。

このデータは、速度指令[m/s]をESC duty比[%]へ変換するための校正値として使う。

## 前提

- PWM周波数は `70 Hz` とする。
- duty比はESCへ出力するPWM duty比[%]である。
- 速度は実車の走行速度[m/s]である。
- 現在のデータは前進方向のみである。
- 後退方向、停止直前、路面差、バッテリー電圧差はこのデータに含まない。

## 実測データ

元データは `scripts/machine.py` のメモから転記した。

| ESC duty比 [%] | 速度 [m/s] |
| ---: | ---: |
| 10.1 | 0.448775 |
| 10.0 | 0.735366667 |
| 9.9 | 1.026866667 |
| 9.8 | 1.235196667 |
| 9.7 | 1.429766667 |

## 近似式

`x` をESC duty比[%]、`y` を速度[m/s]とする。

```text
y = -2.4618 * x + 25.347
```

コードで使う名前にすると次である。

```text
velocity_mps = -2.4618 * esc_duty_percent + 25.347
```

## 逆変換

速度[m/s]からESC duty比[%]を求める式は次である。

```text
x = (25.347 - y) / 2.4618
```

コードで使う名前にすると次である。

```text
esc_duty_percent = (25.347 - target_velocity_mps) / 2.4618
```

## 近似誤差

上記の近似式で計算した速度と実測速度の差を示す。

```text
error_mps = predicted_velocity_mps - measured_velocity_mps
```

| ESC duty比 [%] | 実測速度 [m/s] | 近似速度 [m/s] | 誤差 [m/s] |
| ---: | ---: | ---: | ---: |
| 10.1 | 0.448775000 | 0.482820000 | 0.034045000 |
| 10.0 | 0.735366667 | 0.729000000 | -0.006366667 |
| 9.9 | 1.026866667 | 0.975180000 | -0.051686667 |
| 9.8 | 1.235196667 | 1.221360000 | -0.013836667 |
| 9.7 | 1.429766667 | 1.467540000 | 0.037773333 |

## 使用上の注意

この近似式の実測範囲は次である。

```text
9.7 % <= esc_duty_percent <= 10.1 %
0.448775 m/s <= velocity_mps <= 1.429766667 m/s
```

`0.10 m/s` や `0.30 m/s` のような低速指令は、現在の実測範囲外である。
低速走行確認で使う場合は、ESC中立 `10.55 %` に近い duty比で追加測定する。

追加測定候補:

```text
10.2 %
10.3 %
10.4 %
10.45 %
10.5 %
```

停止指令 `0.0 m/s` は近似式で外挿せず、ESC中立 duty比[%]を直接使う。

現在のESC中立候補:

```text
ESC neutral duty = 10.55 %
```
